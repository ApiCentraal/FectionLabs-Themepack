# FectionLabs Themes

**Twaalf premium color themes. Eén herkenbare signatuur.**

FectionLabs Themes is een complete theme-pack voor developers die lang in de editor werken. Installeer één keer en wissel tussen twaalf moods — van diepe dark themes tot een high-contrast Bright-variant die écht leesbaar blijft.

Werkt in **VS Code**, **Cursor**, **Windsurf**, **VSCodium**, **Trae**, **Antigravity IDE** en elke VS Code-compatible editor.

## Waarom FectionLabs Themes?

- **12 themes in één extensie** — geen losse installs
- **Herkenbare chrome** — champagne-gouden tab-streep (#D4A853), FectionLabs blue (#034972), teal accenten (#0D9488)
- **Syntax voor lange sessies** — keywords, types, strings en errors zijn snel te scannen
- **Bright die écht leest** — navy tekst op wit (geen bleke pastels)
- **Optioneel finetunen** — cursor en line highlight aan/uit in settings

---

## Snel starten

1. Installeer de extensie
2. `Ctrl+K`, daarna `Ctrl+T` (macOS: `Cmd+K`, `Cmd+T`)
3. Kies een **FectionLabs Theme …** variant

Of zet een default in `settings.json`:

```json
"workbench.colorTheme": "FectionLabs Theme Cobalt"
```

---

## Instellingen

| Setting | Default | Beschrijving |
|---------|---------|--------------|
| `fectionlabs.enableCustomCursor` | `true` | Theme-aware cursor kleur |
| `fectionlabs.enableLineHighlight` | `true` | Theme-aware huidige regel |

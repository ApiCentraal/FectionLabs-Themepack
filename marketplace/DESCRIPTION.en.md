# FectionLabs Themes

**Twelve premium color themes. One unmistakable signature.**

FectionLabs Themes is a complete color-theme pack for developers who work long hours in the editor. Install once, switch between twelve curated moods — from deep focus dark themes to a high-contrast Bright variant built for real readability.

## Why install FectionLabs Themes?

- **12 themes in one extension** — no juggling multiple installs
- **Branded chrome** — champagne-gold tab stripe (#D4A853), FectionLabs blue (#034972), teal accents (#0D9488)
- **Syntax tuned for long sessions** — keywords, types, strings, tags, and errors are easy to scan
- **Bright that actually reads** — high-contrast navy text on white (not washed-out pastels)
- **Optional fine-tuning** — toggle custom cursor and line highlight in settings

---

## Quick start

1. Install the extension
2. Press `Ctrl+K`, then `Ctrl+T` (macOS: `Cmd+K`, `Cmd+T`)
3. Choose any **FectionLabs Theme …** variant

Or set a default in `settings.json`:

```json
"workbench.colorTheme": "FectionLabs Theme Cobalt"
```

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `fectionlabs.enableCustomCursor` | `true` | Theme-aware cursor color |
| `fectionlabs.enableLineHighlight` | `true` | Theme-aware current-line highlight |

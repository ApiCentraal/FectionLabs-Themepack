# Compatible editors

FectionLabs Themes works in any **VS Code-compatible** editor.

| Editor | Install |
|--------|---------|
| **VS Code** | [Marketplace](https://marketplace.visualstudio.com/items?itemName=fectionlabs.fectionlabs-theme) |
| **Cursor** | [Marketplace](https://marketplace.visualstudio.com/items?itemName=fectionlabs.fectionlabs-theme) |
| **Windsurf** | [Open VSX](https://open-vsx.org/extension/fectionlabs/fectionlabs-theme) |
| **VSCodium** | [Open VSX](https://open-vsx.org/extension/fectionlabs/fectionlabs-theme) |
| **Trae** | Open VSX or VSIX from [Releases](https://github.com/ApiCentraal/FectionLabs-Themepack/releases) |
| **Antigravity IDE** | [Open VSX](https://open-vsx.org/extension/fectionlabs/fectionlabs-theme) |

Activate: `Ctrl+K` → `Ctrl+T` → **FectionLabs Theme …**

```json
"workbench.colorTheme": "FectionLabs Theme Dark"
```
